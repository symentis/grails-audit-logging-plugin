        • boolean beforeSaveLog(java.lang.Object auditEntity)

          Stub method to modify or override properties prior to save. This is called after all values are set but prior to actually saving.

          Parameters:
          auditEntity - this will be an instance of grails.plugin.auditLog.auditDomainClassName
          Returns:
          true to continue; false will veto the save of the audit entry
